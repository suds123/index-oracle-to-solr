ClickHouse¶

Infrastructure¶

1 VM provisioned on Google Cloud Platform.

  • VM#1
    • Machine type - 24 vCPUs, 64 GB memory
    • CPU platform - Intel Broadwell
    • 150GB SSD persistent disk
    • source image - debian-9-stretch-v20180307

Install ClickHouse¶

Install from debian repository:

sudo apt-add-repository "deb http://repo.yandex.ru/clickhouse/deb/stable/ main/" | sudo tee -a /etc/apt/sources.list
sudo apt-get install dirmngr
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv E0C56BD4
sudo apt-get update

sudo apt-get install -y clickhouse-server clickhouse-client

Start the server:

sudo service clickhouse-server start

Query#2¶

What was the average speed of Yellow taxi trips in 2015 by hour?:

SELECT toHour(pickup_datetime) hour,
       ROUND(AVG(trip_distance / (dropoff_datetime - pickup_datetime))*3600, 1) speed
  FROM yellow_trips
 WHERE trip_distance > 0
   AND fare_amount/trip_distance BETWEEN 2 AND 10
   AND dropoff_datetime > pickup_datetime
   AND pickup_date >= '2015-01-01' AND pickup_date < '2016-01-01'
 GROUP BY hour
 ORDER BY hour;
